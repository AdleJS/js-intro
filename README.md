# JavaScript Introduction

A beginner-friendly repository for learning JavaScript fundamentals, including variables, functions, and basic DOM integration.

## Project Overview

This project contains educational examples demonstrating core JavaScript concepts, designed for developers new to JavaScript.

## Files

### `variables.js`

Demonstrates variable declaration and different data types in JavaScript:

- **`let`** - Block-scoped variable declaration that can be reassigned
- **`const`** - Block-scoped constant declaration
- **Data types** - Examples of `undefined` and `null` values
- **Console logging** - Output variable values using `console.log()`

### `functions.js`

Covers function fundamentals:

- **Function declaration** - Basic function definition and invocation
- **Function parameters** - Passing arguments to functions
- **Return values** - Functions that return and produce results
- **Console output** - Displaying function results

### `index.html`

A simple HTML page that:

- Serves as the entry point for the web application
- Loads `functions.js` using the `defer` attribute
- Displays a heading "My First Javascript App"
- Provides a basic structure for the JavaScript examples

## How to Use

1. Open `index.html` in a web browser
2. Open the browser's Developer Console (F12 or right-click → Inspect → Console)
3. View the output from `functions.js` demonstrating function calls and logging
4. Run `variables.js` separately in Node.js or add it to `index.html` to see variable examples

## Topics Covered

- Variable declaration with `let` and `const`
- Data types and null/undefined values
- Function declaration and invocation
- Function parameters and arguments
- Return values
- Console output for debugging

## Getting Started

To run the JavaScript files directly with Node.js:

```bash
node functions.js
node variables.js
```

To view in a browser, simply open `index.html` in your preferred web browser.
